SSH-managed Linux containers on Proxmox. No signup. Your SSH key is your identity._

VPN Required

Proxbox runs on an internal network. Connect to the OpenVPN gateway before using it.

Request VPN access: lukman@cnrg-labs.id

Quick Start
1
Connect to VPN
$ sudo openvpn --config cnrg-labs.ovpn
2
Create a container
$ ssh -p 2222 proxbox.cnrg-labs.id create dev
Creating container 'dev' (size: small)...
Container 'dev' created (VMID: 5000)
Connect with: ssh proxbox ssh dev
3
Connect to it
$ ssh -tt -p 2222 proxbox.cnrg-labs.id ssh dev
Starting container...
Connecting to dev (10.x.x.x)...

root@proxbox-dev:~# _

Features
Pause / Resume
Containers pause when you disconnect and resume exactly where you left off when you reconnect.
No Signup
Your SSH key fingerprint is your identity. First connection auto-creates your account.
Proxmox LXC
Lightweight Linux containers with near-instant startup. Ubuntu 24.04 LTS with root access.
IDE Support
Works with VS Code Remote SSH, Zed, and any editor that supports SSH remotes.
SSH Config

Add this to ~/.ssh/config to simplify usage:

Host proxbox
  HostName proxbox.cnrg-labs.id
  Port 2222
  RequestTTY force

Then just:

$ ssh proxbox create dev
$ ssh proxbox ssh dev
$ ssh proxbox list
